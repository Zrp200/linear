# linear graph
A gem to graph lines on the first quadrent
## Installation

| Method |Code|
|:------:|:---------:|
|`Gemfile`|`gem 'linear_graph'`
|Terminal|`$ gem install linear_graph`|
|`.gemspec`|`gem.add_runtime_dependency "linear_graph"`|
## Usage
`$ linear_graph slope y-intercept [--zero]`

`$ linear_graph --help`
